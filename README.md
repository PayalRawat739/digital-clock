# Digital Clock ⏰

## 🔹 Description
A simple digital clock application built using Python and Tkinter. It displays the current time in real-time with a clean user interface.

---

## 🛠️ Technologies Used
- Python
- Tkinter

---

## ✨ Features
- Displays real-time time
- Auto-updates every second
- Simple and user-friendly GUI

---

## 🚀 How to Run
1. Install Python
2. Run the Python file:
   ```bash
   python digital_clock.py
